# 🌉 Token Bridge (Soroban Smart Contract)

## 📌 Project Description

Token Bridge is a basic Soroban smart contract built on the Stellar network that simulates cross-chain token transfer functionality. It allows users to deposit and withdraw tokens while maintaining a contract-based balance system.

This project demonstrates the core concepts of smart contract development on Stellar using Soroban, including storage, authentication, and transaction handling.

---

## ⚙️ What it does

* Allows users to deposit tokens into the smart contract
* Stores token balances securely on-chain
* Enables users to withdraw tokens
* Provides a function to check total stored balance

---

## ✨ Features

* 🔐 Secure authentication using `require_auth()`
* 💰 Deposit and withdraw functionality
* 📦 Persistent on-chain storage
* ⚡ Simple and beginner-friendly Soroban implementation
* 🧪 Ready for testnet deployment
